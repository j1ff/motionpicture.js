motionpicture.js

A jquery plugin utilizing sprites to make animations

About

MotionPicture.js is a simple jquery plugin that allows you to use sprites to create animations. It attempts to be flexible for those who have special requirements, while providing sensible default parameters for simple use cases.

Features:

  • All parameters can be scalar values or use callback functions
  • Support for using using class names or background position offsets for changing sprite position
  • Flexible event configuration - You can define any event with any target that should trigger animation
  • Configurable animation start/end logic

How it Works

It's pretty simple. At its core, there are just a few key parameters (each can be either a scalar or a callback function):

  • Focal Point: Determines what percentage of the animation is completed
  • Start Logic: Marks where the animation should begin
  • End Logic: Marks where the animation should end
  • Animation Steps: The number of frames that exist in the animation

With this information in hand, whenever an event that is configured to advance the animation is triggered we determine if the "Focal Point" is within the bounds of the "Start/End Logic", and if it is we find the percentage completion, and assign the appropriate animation step to the element the plugin is modifying.
